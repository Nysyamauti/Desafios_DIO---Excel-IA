# Projeto "Organizador de Declaração de Imposto de Renda"

## Introdução
Este projeto excel é um simulador de um aplicativo que serve para organizar a declaração de imposto de renda.

## Menu
<p>O Menu possui três botões que direcionam para as devidas páginas:</p>
<ul>
    <li>Titular</li>
    <li>Informes</li>
    <li>Notas</li>
</ul>


## Tela do Titular
![Titular](./images/1_titulo.png)
A primeira tela contêm dados do titular. O usuário deve fornecer os dados exigidos para garantir sua identidade relacionado a(s) conta(s) bancária(s) (no melhor cenário).

## Tela do Informes
![Informes](./images/2_informes.png)
A segunda tela contêm três tabelas de informação para os bancos que o usuário possui conta. Na parte de seleção de banco, ela deve corresponder os bancos que a área de seleção possui em seu banco de dados, caso ao contrário, irá aparecer um pop-up "não foi possivel encontrar o banco informado". E no topo, possui uma tabela de total que soma os valores monetários presentes de cada conta bancária.

## Tela de Notas
![Notas](./images/3_Notas.png)
A terceira e última tela, envolve Notas de entrada de valores. No cenário proposto, ele guardaria a data, categoria e valor da entrada que foi realizada pelo usuário.
